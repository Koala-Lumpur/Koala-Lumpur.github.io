Replay Football

Description

A Data-based VR training software for football clubs, designed to improve training efficiency and player performance. Coaches and players can replay real football matches from any perspective, or step directly into the action with a VR headset immerse themselves in the match. Built in Unity, by a small development team, with third-party API and data integration from multiple providers.

I contributed to most major features, with a particular focus on VR integration and Multiplayer systems using Netcode for GameObjects. I also created automation tools for generating and exporting football player faces from Blender to Unity, streamlining the content pipeline.

One key challenge was optimizing for mobile VR, having to find a constant balance between performance and quality, which includes reducing complexity of models, minimizing draw calls and optimizing code, all while maintaining a visually immersive experience.

Interface for Spatial Audio Mixing in Virtual Reality

Description

A university group project (5 members) focused on building an immersive 3D audio mixing interface in VR. The system integrated directly with Ableton Live, allowing users to mix audio in real time inside a 3D environment.

I served as the main Unity developer, implementing VR interactions and core functionality for the interface. I also optimized models for performance on the Oculus Quest headset and contributed some 3D assets.

A key challenge was enabling reliable communication between Unity and Ableton Live. I solved this by implementing UDP and OSC protocols, allowing the VR headset to send and receive real-time audio control signals from the computer.

Imitation of Natural Settings in Virtual Reality for Relaxation

Description

A university group project (5 members) focused on building relaxing VR environments. Users could choose between different natural settings and immerse themselves in atmospheric visuals and spatial audio for a calming experience.

I was the main Unity developer, implementing the core functionality, designing the environments, and ensuring the project ran consistently across devices. I also managed version control and backups using Git.

Puzzle Solving Machine Learning Agent

Description

A solo project where I trained an AI agent in Unity using ML-Agents and TensorFlow. The agent was tasked with solving a simple puzzle: pushing a box onto a pressure plate to unlock a door and reach the exit.

I designed and implemented the environment, defined the agent's possible actions, and structured the reward system to guide learning. This included balancing immediate vs. long-term rewards to encourage optimal behavior.

The biggest challenge was designing an effective reward structure and tuning the environment complexity to reduce training time. Through iteration, I achieved a setup where the agent could consistently learn the optimal solution.
